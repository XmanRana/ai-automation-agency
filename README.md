# AI Automation Agency MVP

Full-stack AI automation platform with modular tool system.

## Features

- **Email Generator** - Create professional emails
- **Document Processor** - Extract and process documents
- **Data Analyzer** - Analyze data patterns and trends

## Tech Stack

- **Backend:** Laravel 11, PHP 8.2
- **Frontend:** React 18, Vite
- **HTTP Client:** Axios
